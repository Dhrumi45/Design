## Circle Design with Turtle Graphics 🌀
This Python script uses the turtle module to draw a colorful spiral of circles and display the text "Circle Design" below. A simple visual design project using loops and turtle graphics.

## Hexagon with Colored Triangles 🎨
This Python script uses the turtle graphics library to draw a hexagon with six colored triangles extending outward from each side. The hexagon is centered on the screen with bold black edges, and each triangle is filled with a different color. A label below the design names the artwork. The code demonstrates geometric drawing, color filling, and positioning using turtle graphics.
